# ctx-dev-setup

Install VS Code, Docker, and core development tools.

## Usage

```bash
ctx-dev-setup
```

## Features

- VS Code installation from official Microsoft repo
- Docker Engine with latest stable release
- Automatic group permissions setup

## Requirements

- Debian Bookworm or later
- Internet connection
- sudo privileges
