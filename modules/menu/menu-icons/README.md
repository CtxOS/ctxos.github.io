# CtxOS Menu Icons

This directory contains icon assets for CtxOS menu applications.

## Directory Structure

```
menu-icons/hicolor/
├── 256x256/apps/     # Master source icons (place your 256x256 PNG icons here)
├── 48x48/apps/       # Generated 48x48 icons (1x)
├── 48x48@2/apps/     # Generated 96x96 icons (2x HiDPI)
├── 32x32/apps/       # Generated 32x32 icons (1x)
├── 32x32@2/apps/     # Generated 64x64 icons (2x HiDPI)
├── 24x24/apps/       # Generated 24x24 icons (1x)
├── 24x24@2/apps/     # Generated 48x48 icons (2x HiDPI)
├── 16x16/apps/       # Generated 16x16 icons (1x)
└── 16x16@2/apps/     # Generated 32x32 icons (2x HiDPI)
```

## Usage

1. **Add Master Icons**: Place your 256x256 PNG icons in `256x256/apps/`
   - Use descriptive names matching your desktop files (e.g., `ctxos_nmap.png`)
   - Icons should be 256x256 pixels, PNG format

2. **Generate Multi-Resolution Icons**: Run the build script:
   ```bash
   python3 build-icons.py
   ```
   This will automatically generate all the smaller sizes and HiDPI variants.

3. **Install**: Icons are automatically installed to `/usr/share/icons/hicolor/` when the menu module is installed.

## Icon Naming Convention

- Tool icons: `ctxos_<toolname>.png` (e.g., `ctxos_nmap.png`)
- Category icons: `<category-name>.png` (e.g., `info-gathering.png`)
- Transparent variants: `<name>-trans.png` (e.g., `info-gathering-trans.png`)

## Notes

- The `256x256/apps/` directory is the **master source**
- All other directories are auto-generated by `build-icons.py`
- HiDPI (@2) variants are 2x the base resolution
- Icons are installed during `make install` or when the menu module is deployed
